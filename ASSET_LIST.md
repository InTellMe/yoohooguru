## Prompt Style Guide

### Base Style Directive
```
Professional Sasquatch character illustration, Pixar/DreamWorks style, warm and approachable, 
soft lighting, Tennessee mountain backdrop, modern technology integration, 
family-friendly, high detail, 8K resolution, slight bokeh background
```

### Color Palette
- **Primary:** Forest greens (#10B981, #059669)
- **Secondary:** Mountain blue (#3B82F6, #1E40AF)  
- **Accent:** Warm amber (#F59E0B)
- **Background:** Misty grays and warm wood tones

## Generation Services

Recommended AI image generation services:
1. **Midjourney** - Best for character consistency and Pixar-style rendering
2. **DALL-E 3** - Good for quick iterations and variations
3. **Stable Diffusion XL** - Best for local generation and customization

### Consistency Tips
- Generate a "character sheet" first with front/side/back views
- Use seed values to maintain consistency across images
- Create a LoRA or fine-tuned model for each character for best results
