# Sasquatch Hero Image Prompts (Subdomain Pages)

Use these prompts to generate hero images for each subdomain. Upload each image to
Cloudinary folder `dksj2niho` with the matching public ID below (no file extension).

Base style (append to each prompt):
Photorealistic, professional Sasquatch, warm natural light, soft shadows,
cinematic color grading, shallow depth of field, high detail, 8k.
